    ## PHPとは
    <img src="/phase1/website/hello-world/helloworld.png" loading="lazy" alt="Photo by Minh Pham" class="" />
    PHPは1995年にRamsmus Lerdorf氏により『Personal Home Page Tools』という名前で開発されました。
    その後改良されて現在のPHPとなりました。PHPはスクリプト型のサーバーサイドスクリプト言語となります。
    サーバーサイドスクリプト言語はプログラムがHTML(Webページの元ファイル)の中に埋め込まれて
    Webサーバー上で実行されてWebページを返します。

    ### PHPの特徴
    PHPの特徴としてサーバーサイドスクリプト言語のため、DB操作が行えるということが
    大きな特徴として挙げることができます。その他にもWordPressでPHPが採用されていることにより
    たくさんのコーポレートサイトやECサイトの制作時にも採用されている言語です。

    ### PHPを学習するメリット

    PHPのメリットとして
    ・初心者におすすめ
    ・学習教材・開発実績が豊富
    ・データベースとの連携が容易
    ・構文がシンプルで分かりやすい
    が挙げることができます。

    ### 初心者におすすめ

    PHPは構文がシンプルで、他のプログラミング言語と比較すると「変数の型を意識しなくていい」
    「配列の定義が自由」など制約が少ないのが特徴です。そのためPHPは学習難易度が低く、初心者が習得がしやすいプログラミング言語の一つと言われています。
    PHPはC言語やJavaなどと構文が似ていると言われているため、
    まずは難易度の低いPHPを習得してプログラミング言語を学習するコツをつかんでから他言語の勉強を進めるのもおすすめですよ。


    ### 学習教材・開発実績が豊富
    PHPは人気かつ、長期間第一線で活躍するプログラミング言語です。
    そのためPHPに関するWebサイトや書籍などの日本語教材が豊富にあり、学習時に分からないことがあっても解決しやすいという特徴があります。
    さらにノウハウやオープンソースのプログラムも多く公開されているため、効率的な開発が可能です。

    ### データベースとの連携が容易

    PHPはMySQLなどのデータベースとの連携が得意なので、情報をデータベースに保存・更新したり、大量のデータの中から必要なデータを瞬時に取り出すことができます。
   さらにPHPはHTMLで記載したコードに直接組み込むことができるため、Webページ上にサーバーにある情報をWebサイト上に表示することも可能。この機能はECサイトの
   「商品検索機能」や飲食店サイトの「予約機能」などに活用されています。

    ### Hello Worldと出力してみよう!
    まずはPHPを使ってHello,World!とブラウザに出力させてみましょう。

    #### なぜHelloWorld?
    プログラミング言語を自身のPCで問題なく動くか試す時にまずはブラウザ等にHelloWorldと出力させて
    みるという伝統？のようなものがあります。正直文字や記号を出力が確認できれば何でもいいのですが
    大抵のプログラミング言語の入門書では、このプログラムを作ることを最初の例題としており、
    新しくプログラミング言語を習得する際に最初に作るのがこのプログラムです。

    ※ここでは深掘りしませんが気になる方は講師に聞いてみたり、自身で調べてみるのも面白いかもしれません。

    index.html

    ```
    <!DOCTYPE html>
    <html lang="ja">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>タイトル</title>
    </head>

    <body>
    <script src="main.js"></script>
    </body>
    </html>
    ```

    ほとんど雛形のままですが、ハイライトしてある行が重要です。この行がHTMLからPHPを呼び出している部分になります。scriptタグのsrc属性に、呼び出したいPHPのファイル名を書きます。
    同じフォルダに、以下のようなmain.jsを追加してください。
    main.js
    console.log('Hello, world!')
    ここまで準備ができたら、index.htmlをChromeで開いてください。
    
    何も表示されませんが、裏ではしっかりPHPが実行されています。Chromeの「デベロッパーツール」を
    開いてください。右上のメニューの [その他のツール] → [デベロッパーツール] で開くことができます。
    上部の [Console] というタブを選択して「コンソール」という画面を開きます。すると、ここにHello, world!と
    表示されているはずです。
    
    これが初めてのPHPプログラムができました!
    Step1 console.logとコンソール
    console.log(出力したい内容)
    のように書くと、コンソールにその内容が表示されます。
    コンソールは、普段の人は開かないので、実案件で作るようなプログラムの出力先には向いていませんが、プログラマが表示確認する意図で多く使われます。この講座でも、しばらくはconsole.logを使って表示確認をしていきます。
    console.logを使って、いろいろな出力をしてみましょう。
    コンソール上で、

    ```

    'Hello, world!','こんにちは世界！'、1+2,10 - 4 * 8

    ```

    #### コメント/コメントアウト

    上記の例で、/* いろいろな例 */や// 日本語もOKと書いてあるところに注目してください。
    PHPでは、/*と*/で囲まれた間、もしくは//からその行の最後まではプログラムだと解釈されません。
    これらを一般的には「コメント」と呼びます。人間が読んでわかりやすいコメントを残すために使われたり、
    コードを一時的に無効化するのに使われます。また、このようにコメント化することをコメントアウトと呼びます。
