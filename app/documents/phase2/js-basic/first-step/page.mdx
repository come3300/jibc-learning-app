    プログラミング言語とマークアップ言語は違うので

    文法などの書き方だけではなく、その変化に対応していける基礎力をつけるという事も意識して進めていってください！

    ### JavaScriptとは

    JavaScript（ジャバ スクリプト）は、
    動的なWebページを作るためのプログラミング言語の一つです。
    1996年頃、Internet Explorer 3.0に搭載されたことをきっかけに、急速に普及しました。

    代表的な機能は、ボタンを押すと画面に別ウィンドウが開く「ポップアップウィンドウ」や、
    画像が自動送りされる「スライドショー（スライダー）」などの多様なアニメーションです。
    その他、システムやアプリ開発などのバックエンドでも用いられています。

    #### JavaScriptとJavaの違い

    よく初心者の方がする間違いとしてJavaScriptとJavaが兄弟分のように
    考える方がいますが、全くの別物と考えてください。
    （メロンとメロンパンくらいの違いがあります）

    そしてJavaScriptを略してJavaと呼ぶ方がいらっしゃいますが、それも間違いです。
    JavaScriptを略して読むときはJS（ジェーエス）と呼ぶのが正しい略し方になります。

    ### JavaScriptを学習するメリット

`
JavaScriptのメリットとして >・初心者におすすめ >・環境構築が簡単 >・Webサイトに動きをつけられる >・非同期処理が行える >・サーバーサイドも対応できる

    が挙げることができます。

    #### 初心者におすすめ

    まずJavaScriptのメリットとして初心者がとっつきやすいプログラミング言語であることが挙げられます。
    JavaScriptは遊び心のあるプログラミング言語なので様々なゲームやアプリ、システムを作りながら
    プログラミングの書き方や文法、法則を学ぶことができるため一通り書き方を学んだら自身の作りたいものを
    自身で作ってみると良いでしょう。

    #### 環境構築が簡単

    JavaScriptは他のスクリプト言語（php、python,ruby）と違ってクライアントサイトの言語になるため
    環境構築が非常に簡単であり、非常に初心者向きの言語といえます。

    #### Webサイトに動きをつけられる

    皆さんはWebサイト上でアニメーションや動きのあるサイトを閲覧したことはないでしょうか？
    こういった動きはJavaScriptで実現可能になります。（CSSでもアニメーションは実装可能です。）
    JavaScriptを学習することでよりリッチなWebサイトの制作が可能になります。

    #### 非同期処理が行える

    JavaScriptには「非同期処理」を行うための機能が標準搭載されています。
    非同期処理とは、Webページ全てを読み込まずに必要な部分だけを読み込むことができる仕組みのことです。

    皆さんの馴染みのあるものとして『GoogleMap』が挙げられます。
    飲食店のサイト等でそのお店の場所を『GoogleMap』で埋め込み表示されていることがあります。
    この仕組みを実現するのに非同期処理が使われています。

    非同期処理を行わずに『GoogleMap』を同じように操作しようとなると一度皆さんのPCに
    『GoogleMap』のデータをダウンロードしなくてはいけません。
    これだとPCのストレージ（容量）を圧迫してしまいダウンロードにも時間がかかってしまい大変です。
    こういった非同期処理を搭載されることでより私たちの生活が便利になっているのです。

    #### サーバーサイドも対応できる

    Webシステムは、ユーザーから見える部分の「フロントエンド」、裏方部分の「サーバーサイド」の2要素で構成されます。
    （サーバーサイトでは主にデータベース上の操作やhttpレスポンス上でのエラー処理や認証処理等を行います。）
    ブラウザ上で動作するJavaScriptはフロントエンドの開発言語ですが、サーバーサイドの開発も可能です。
    前述のように、JavaScriptのプログラムは基本的にブラウザ上で動作します。
    しかし、「Node.js」というソフトウェアを使うことで、Webサイトのデータを管理する「Webサーバー」上でも
    動作が可能となります。そのため、JavaScriptを習得すればフロントエンド開発も、
    サーバーサイド開発も可能です。JavaScriptを身につけるだけで、Webアプリ開発の選択肢はかなり広がります。


    ### Hello Worldと出力してみよう!

    まずはJavascriptを使ってHello,World!とブラウザに出力させてみましょう。

    #### なぜHelloWorld?

    プログラミング言語を自身のPCで問題なく動くか試す時にまずはブラウザ等にHelloWorldと出力させて
    みるという伝統？のようなものがあります。正直文字や記号を出力が確認できれば何でもいいのですが
    大抵のプログラミング言語の入門書では、このプログラムを作ることを最初の例題としており、
    新しくプログラミング言語を習得する際に最初に作るのがこのプログラムです。

    ※ここでは深掘りしませんが気になる方は講師に聞いてみたり、自身で調べてみるのも面白いかもしれません。

    ```html
    <!DOCTYPE html>
    <html lang="ja">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>タイトル</title>
    </head>

    <body>
    <script src="main.js"></script>
    </body>
    </html>
    ```

    ほとんど雛形のままですが、ハイライトしてある行が重要です。
    この行がHTMLからJavaScriptを呼び出している部分になります。
    scriptタグのsrc属性に、呼び出したいJavaScriptのファイル名を書きます。
    同じフォルダに、以下のようなmain.jsを追加してください。
    //main.js
    ```JavaScript
    console.log('Hello, world!')
    ```
    ここまで準備ができたら、index.htmlをChromeで開いてください。

    何も表示されませんが、裏ではしっかりJavaScriptが実行されています。Chromeの「デベロッパーツール」を
    開いてください。右上のメニューの [その他のツール] → [デベロッパーツール] で開くことができます。
    上部の [Console] というタブを選択して「コンソール」という画面を開きます。すると、ここにHello, world!と
    表示されているはずです。

    これが初めてのJavaScriptプログラムができました!
    Step1 console.logとコンソール
    console.log(出力したい内容)
    のように書くと、コンソールにその内容が表示されます。
    コンソールは、普段の人は開かないので、実案件で作るようなプログラムの出力先には向いていませんが、
    プログラマが表示確認する意図で多く使われます。この講座でも、
    しばらくはconsole.logを使って表示確認をしていきます。
    console.logを使って、いろいろな出力をしてみましょう。
    コンソール上で、

    ```
    'Hello, world!','こんにちは世界！'、1+2,10 - 4 * 8
    ```

    ***

    #### コメント/コメントアウト

    上記の例で、/* いろいろな例 */や// 日本語もOKと書いてあるところに注目してください。
    JavaScriptでは、/*と*/で囲まれた間、もしくは//からその行の最後まではプログラムだと解釈されません。
    これらを一般的には「コメント」と呼びます。人間が読んでわかりやすいコメントを残すために使われたり、
    コードを一時的に無効化するのに使われます。また、このようにコメント化することをコメントアウトと呼びます。
