    ## サーバーについて
    ***
    <img src="/phase1/website/hello-world/helloworld.png" loading="lazy" alt="Photo by Minh Pham" class="" />

皆さんが一人前のエンジニアとして活躍している数年後、あるお客さんから
『会社のサイトを作ってほしい！』と依頼が来たとします。

ここで考えなくてはいけないのは、サイトを制作するには
どんなものが必要になってくるかになります。

    例えば一般的なコーポレートサイトにはこんなものを準備しなくてはいけません

    >・メールを送信するためのサーバー
    >・Webを表示させるためのサーバー
    >・ユーザーの情報を保存しておくためのサーバー

前回の講義インターネットの仕組みでネットを表示させるためにはサーバーにリクエストしてレスポンス
を返してもらう必要があると教えましたが、サイトを動かすにはメールのためのサーバーや
ユーザーの情報を保存しておくためのサーバーなどが必要になってきます

### サーバーの種類

ではサーバーにはどんなものがあるのでしょうか。
これから皆さんがエンジニアとして活躍するためには
下記のサーバーを理解しておく必要があります。

・Webサーバー
・メールサーバー
・データベースサーバー
・DNSサーバー
・FTPサーバー

### Webサーバー

WebサイトやWebアプリを運営する際に必要なサーバーです。
利用者の要求に応じてデータをWeb上に送り返す役割があります。

HTML/CSSを使って表示されるためのデータを管理する役割もあります。
そして重要なのがブログやWebサイトはWebサーバーがないと表示できません。
Web開発においてWebサーバーは非常に大きな役割を持っています

例
・Apache
・Nginx

### メールサーバー

メールサーバには大きく分けて二つのサーバーがあります。

-画像-

・送信を担当するSMTPサーバー

-画像-
・受信を担当するPOPサーバー

#### SMTPサーバーとは

-画像-

Simple Mail Transger Protocolの略称になります。
送信したメールを指定のアドレス先にしっかり送り届けるのが役割です。

#### POPサーバーとは

-画像-

Post Office Protocol Serverの略称です。
送られてきたメールを認証して受信する役割があります。

### データベースサーバー

-画像-

データベースサーバーは利用者の要求に応じてデータベース上のデータを送り返すための
サーバーになります。

※データベースサーバーとWebサーバーは別物ですので
違いをしっかりと理解しましょう。

例えば皆さんが普段図書館に行ったときに自分の読みたい本を
探すときに書籍の検索をPCで行ったことはありませんか。

これは書籍が保存されているDB上からほしい本を検索して
探し出しているのです。

### DNSサーバー

-画像-

DNSサーバーはDomain Name System Serverの略称です。
Webサイトなどの利用者の要求意応じてIPアドレスとドメインを結び付ける役割があります。

#### IPアドレスとは

-画像-

IPアドレスとはWeb上の住所のようなものです。
例えば172.217.24.131のように数字が羅列しています。

-画像-

このIPアドレスだけではわかりづらいためこのIPアドレスに名前をつけて
わかりやすくしたのがドメインになります。

Webサイトにアクセスするためには特定サイトのWEbサーバーを見つけ出し
IPアドレスとドメインを結び付ける必要があります。

-画像-

DNSサーバーは特定サイトのWebサーバーを見つけ出した上で
IPアドレスとドメインを照合してWebサイトが閲覧できるようにする役割があります。

-画像-

つまりDNSサーバーがあることでWebサーバーから特定のIPアドレスを見つけ出して
そのIPアドレスとドメインを結びつけてくれるということになります。

### FTPサーバー

-画像-

File Transfer Protocol Serverの略称で
ファイル転送プロトコルともいわれてます。

FTPサーバーは下記のことを主にしてくれます。

・Webサイトの更新
・Webサイトのデータ上書
・ファイルのアップロード
・ファイルのダウンロード

FTPサーバーはFTPがファイルの送受信をする際のルールに乗っ取って
ファイルの送受信を可能にします。

WebサイトからPDFをダウンロードした経験はありませんか？
このPDFをダウンロードする際にFTPサーバーを使ってPDFのダウンロードが行われています。

Web上でファイルの送受信をする際にFTPサーバーが活躍してくれると認識していればOKです。
