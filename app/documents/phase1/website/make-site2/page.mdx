        HTMLができたらinnerのCSSを作っていきましょう。
        先ほど軽く説明しましたが、innerは画面サイズが変わっても中央に表示させるためのもので、

        次はnewsセクションでも使うタイトルクラスのCSSを作成しましょう。
        このタイトルの横にある線は擬似要素を用いて作成します。
        擬似要素で線を作る方法はいくつかありますが、今回は擬似要素にborderをつけて実装していきます。
        titleとtitleの擬似要素2つを::befroreと::afterで作ります。
        そこにwidthを指定して```border-top```を付ければ線が出来上がります。
        文字との間を作るためにwidthも追加しておきましょう。
        擬似要素```（::before,::afeter）```は後で横並びに、
        かつ横幅を指定しなければいけないので、inline-block要素にすることを忘れないようにしましょう。

        ```css
        /*共通パート*/
        .title {
            font-size: 18px;
            /* 文字をイタリック体にイタリック体 */
            font-style:italic;
        }

        .title-b{
            /* Aboutの部分だけを太文字に */
            font-weight: bold;
        }

        .title::before{
            /* 擬似要素には必ずつける */
            display:inline-block;
            width:30px;
            margin-right:20px;
            border-top:1px solid #191919;
        }

        .title::after {
            /* 擬似要素には必ずつける */
            content:"";
            display:inline-block;
            width:30px;
            margin-left:20px;
            border-top:1px solid #191919;
            content:"";
        }
        ```

        デザイン部分ができたら要素を全て横並びにして、
        線を文字中央に持ってこれるようにしましょう。titleと擬似要素（::before,::after）の3つの要素を
        display:flex;で横並びにすることで、titleの横に線を作ることができます。
        また横並びにできたら```align-items:center;```で線を文字の縦方向中央に配置できるようにしましょう。

        ```css
        /*共通パート*/
        .title{
            /* 擬似要素とtitleを横並びに */
            display:flex;
            padding-bottom:60px;
            /* 要素全体を横方向に中央揃え */
            justify-content: center;
            /* 線を文字の縦方向に中央揃え */
            align-items: center;
            font-style:italic;
            font-size: 18px;
            text-align:center;
        }
        ```

        タイトルのスタイルが完成したら、about-itemが綺麗に並ぶようにCSSを書いていきましょう。
        ここでabout-wrapクラスにの中に内包している6つの子要素を横並びにするために、```display:flex;```を
        つけるだけでは全て横並びになってしまい、親要素からaboutーitemが飛び出してしまいます。
        そうならないように
        * 子要素が折り返されるように.about-wrapクラスでflex-wrap:wrapをつける
        * 内包されているクラスにwidthを指定するようにしましょう。
        またaboutセクションの4つ目の子要素が1-3個目の子要素にくっついてしまうので、
        :nth-child()でクラス指定でmarginで要素がくっつかないようにしましょう。
        今回使う::nth-child(n+4):nth-child(-n+6)では
        4~6個目のabout-itemを指定していることになります。

        ```css
        /* aboutのスタイル */
        #about{
            color:#191919;
            background-color:#F8F8F8;
        }

        .about-wrap{
            /* 子要素を横並びに */
            display:flex;
            /* 横幅が親要素を超える時に子要素を次の段に移す */
            flex-wrap: wrap;
            margin-top:60px;
        }

        .about-item{
            width:300px;
            height:315px;
            padding:35px 50px 50px;
            text-align: center;
            background-color:#FFF;
        }

        .about-item:nth-child(n+4):nth-child(-n+6){
            /* 4~6個目のabout-itemにmarginをつける */
            margin-top:47px;
        }

        .about-title__title{
            font-size: 22px;
        }

        .about-item__img img{
            width:122px;
            height:122px;
            margin:20px 0 25px;
        }

        .about-item__desc{
            font-size: 13px;
        }
        ```

        プルダウン型の選択式メニューであるセレクトボックスを作ります。
        selectタグに元々ついている矢印アイコンを消し、矢印アイコン画像を貼ることでデザインを綺麗にしていきます。
        nth-of-type()プロパティと擬似要素を使って、矢印アイコン画像を実装していきます。

        ```css
            position:absolute;
            background:url("../img/arrow.png");

            height:15px;
            top:60%;
            right:15px;
        }
        ```

        ```css
        }
        ```
