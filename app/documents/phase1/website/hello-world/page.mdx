    ### Hello Worldとは？
    Hello Worldとは？環境構築が完了した上でブラウザ上に
    しっかりと出力できるかを確かめるための手段です。

    場合によってはこれが最初の関門だったりします。
    HTML/CSSの場合はそこまで難しくはありませんが、システムやアプリを作成するとなると
    より難しくなります。

    ### ディレクトリを作成しよう

    ではHTML/CSSを出力できるようにするためにディレクトを
    作りましょう。

    #### GUIで作成してみる

    まずはGUIでの作成していきましょう。
    場所はどこでもいいので

    1,新規フォルダを作成>helloという名前のディレクトリを作成

    2,helloディレクトリ内にhello.htmlというファイルを作成する

    下記画像のようになっていれば問題ないです。

    #### CUIで作成してみる
    
    こちらは余裕がある人がやってみてもらえればと思います。
    まずはGUIの方で作成できれば問題ありません。

    ```
    $ mkdir hello 
    //helloディクレトリの作成

    $ cd hello
    //helloに移動する

    $ touch hello.html
    hello.htmlというファイルを作成する
    
    yonekuradaisuke@yonekuradaisukenoMacBook-Pro-2 hello % tree

    hello
    └── hello.html
    //このようになっていれば問題なく作成できています。

    ```

    ### Hello Worldの出力方法    

    では、ディレクトリの作成が完了したので
    HelloWorldと出力していきましょう。

    まずはVSCを開いてhelloディレクトリを
    ドラックアンドドロップしましょう。

    次にhello.htmlに下記のコードを書きます。

    ```
    <p>Hello World</p>
    ```

    そしてhello.htmlをブラウザに
    ドラックアンドドロップしてみましょう。

    <img src="/phase1/website/hello-world/helloworld.png" loading="lazy" alt="Photo by Minh Pham" class="" />

    ブラウザに下記画像のようにHello Worldと表示されれば成功です。
