        ### spのセットアップ

        sp用のファイルにメディアクエリを追加していきます。

        ```css
        /*-------------------------------------------
        SP
        -------------------------------------------*/
        @media screen and (max-width: 600px) {

        /*-------------------------------------------
        hero
        -------------------------------------------*/
        #mainvisual img {
            height: calc(100vh - 60px);
        }

        }

        ```

        今回のSassを用いたレスポンシブ化では画面の幅がスムーズに切り替わるよう、
        font-sizeをpx単位ではなく、親要素のサイズによって文字サイズが変わるem,remを用いて記述していきます。
        em,remでも単位がわかりやすくなるように_lib.scssにhtml,bodyのスタイルを追加します。
        remは最上位のHTML要素である「HTML」の文字サイズを基準とした相対値で文字サイズを決められるので、
        html要素にfont-size:62.5%;と指定して10px=1remになるように設定しましょう。
        emも同様に相対要素ですが、基準となる要素が親要素となります。

        ```css
        /* 共通部分 */
        html{
            font-size: 62.5%; //10px=1remになる
        }

        body{
            font-size:12px;
            line-height:1.67;
        }
        ```

        ### aboutのsp実装

        <img src="/phase1/website/responsive/about-sp.png" loading="lazy" alt="Photo by Minh Pham" class="" />

        aboutのsp実装をしていきます。
        pc幅との違いとして画像が横並びから縦並びになっているのがわかると思います。

        こちらを実装するために必要になってくるのが

        ```css
        flex-direction: column;
        ```

        になります。このセレクタを実装することにより横長びになっている画像が
        縦一列に並び変えることが可能です。

        <img src="/phase1/website/responsive/flex-figure.png" loading="lazy" alt="Photo by Minh Pham" class="" />

        この状態では余白が整っていないため
        marginで整えます。

        ```css
        /*-------------------------------------------
        About
        -------------------------------------------*/
        #about .content {
            flex-direction: column;
        }
        #about img {
            margin-right: 0;
        }

        ```

        ### blogセクションのsp実装

        <img src="/phase1/website/responsive/blog-sp.png" loading="lazy" alt="Photo by Minh Pham" class="" />

        先ほどと同様に flex-direction: column;を使い縦一列に画像が
        並ぶようにしましょう。

        画像の幅と位置を調整すれば完了です。

        ```css
        /*-------------------------------------------
        Bicycle
        -------------------------------------------*/
        #bicycle ul {
            flex-direction: column;
        }
        #bicycle li {
            width: 100%;
            margin-bottom: 30px;
        }

        ```

        ### sp全文

        ```css
        @media screen and (max-width: 600px) {

        -------------------------------------------*/
        #bicycle ul {
            flex-direction: column;
        }
        #bicycle li {
            width: 100%;
            margin-bottom: 30px;
        }
        }

        ```
